# Installation

## From crates.io

Build from published source:

```
$ cargo install heretek --locked
```

## GitHub Releases

Download pre-built binaries from [GitHub releases](https://github.com/wcampbell0x2a/heretek/releases).

## Arch Linux

Install from the [official repositories](https://archlinux.org/packages/extra/x86_64/heretek/):

```
$ pacman -S heretek
```
